gmax Registration Policy

The following text describes the gmax (free version) registration process and Discreet's policy on gmax registration information.

gmax (free version) requires on-line web registration using IE5.0 or greater before it can be used.

There is currently no support for registering gmax (free version) on machines that cannot browse the web via IE 5.0 or greater.

New gmax users must visit www.discreet.com/products/gmax/register using each machine from which they intend to run gmax, and complete the on-line web registration process.

Registration of gmax on multiple PCs for a single user can be accomplished using the registration password that is automatically emailed to a new gmax registrant after their initial gmax on-line web registration. Entering the identical email address and the e-mailed gmax password on the gmax registration web form during successive installations will generate a valid registration that is identical on all machines. A valid e-mail address is required during initial registration to ensure receipt of the password and to register multiple installations of gmax per user.

During gmax on-line web registration, as with many web sites that utilize login or registration information, a browser "cookie" is written by Discreet's servers onto each user's PC, containing basic gmax registration information. Afterwards, during normal use, gmax reads from the browser cookie at launch to determine registration status. Proper registration information in the browser cookie allows gmax to run without delay.

Because gmax relies on browser cookies for initial registration, "cookie cleanup" (deletion of browser cookies by a user) will affect gmax operation. If a user removes gmax registration browser cookies after a valid on-line web registration session, gmax will continue to operate, but it will post a passive notice each time it is launched, reminding the user that their registration is incomplete. gmax will continue to run normally in this case. To repair an incomplete registration, gmax on-line web registration provides for easy recovery: by using the gmax registration password sent to each new gmax registrant's e-mail address during initial gmax on-line web registration, a gmax user can quickly regenerate a valid browser cookie on any internet-connected machine (as described above for multiple PC registrations).

Discreet collects gmax registration data for statistical use and to help facilitate gmax user services and future product developments. gmax registration data is a vital part of Discreet's ability to offer gmax to consumers for no charge. Discreet respects gmax user privacy and individual preferences for the use of gmax registration data.
